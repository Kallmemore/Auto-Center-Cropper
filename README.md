Auto Center Cropper

Auto Center Cropper is a Python GUI app for center-cropping images with custom resolution. Supports batch processing, drag & drop, and shows progress with COMPLETED! on finish.

Requirements

Python 3.8+

Pillow

tkinterdnd2

Install dependencies:

pip install pillow
pip install tkinterdnd2


Usage

Run the app:

python "Auto Center Cropper.py"


Drag files/folders into the app or onto the app icon to process.
